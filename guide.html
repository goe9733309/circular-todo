📖 사용법

둥근 24시간 시간표의 모든 기능을 순서대로 안내합니다.

1. 일정 추가하기

방법 A · 직접 입력하기

  1. 시작 시각과 종료 시각을 입력합니다.
  2. 할 일 칸에 이름을 적습니다. (예: 운동)
  3. 원하는 색상 스와치를 고르거나, 맨 끝의 동그란 버튼으로 직접 색을 지정합니다.
  4. 필요하면 알람 설정을 켜고, 시작/종료 시각 중 언제 울릴지 고릅니다.
  5. 추가 버튼을 누르면 시계 위에 밴드가 생깁니다.

방법 B · 시계판에서 드래그하기

  1. 아직 일정이 없는 빈 시계판 위를 마우스로 누른 채 원하는 방향으로 드래그합니다.
  2. 손을 놓으면 시작 시각 / 종료 시각이 10분 단위로 반올림되어 자동으로 채워지고, 선택한 영역은 점선 밴드로 계속 표시됩니다.
  3. 마음에 들지 않으면 밴드 바깥의 ✕ 표시를 눌러 취소할 수 있습니다.
  4. 새로 드래그하면 이전 영역은 자동으로 대체됩니다.
  5. 할 일 이름을 입력하고 추가를 누르면 정식 일정으로 등록됩니다.
💡 이미 일정이 있는 자리를 클릭하면 그 일정의 툴팁이나 이동/회전 손잡이가 우선 동작하므로, 드래그로 새 일정을 만들 때는 빈 공간에서 시작하세요.

2. 일정 삭제하기

  • 왼쪽 오늘의 할 일 목록에서 해당 항목의 삭제 버튼을 누릅니다.
  • 또는 시계 위 일정 라벨에 마우스를 올리면 나타나는 ✕ 버튼을 눌러도 바로 삭제됩니다.

3. 라벨 위치·모양 바꾸기

시계 위 일정 이름(라벨)에 마우스를 올리면 손잡이가 나타납니다.

  • 라벨을 직접 드래그하면 위치를 옮길 수 있습니다.
  • ↻ 손잡이를 드래그하면 글자가 회전합니다.
  • ↕ 손잡이를 드래그하면 글자 크기가 커지거나 작아집니다.
  • 손잡이를 더블클릭하면 각각 원래 위치·각도·크기로 되돌아갑니다.

4. 표시 설정

표시 설정 카드를 펼치면 시계의 생김새를 바꿀 수 있습니다.

화면 테마

시스템 설정 따르기 / 라이트 / 다크 중에서 고를 수 있습니다.

시계판 모양

도넛형(기본) · 얇은 밴드형 · 파이형 · 아웃라인형 네 가지 스타일이 있습니다.

시계 글자 폰트

고딕체부터 손글씨, 붓글씨 느낌까지 다양한 글꼴로 바꿀 수 있습니다.

시계 제목 · 상단 표시

시계 위쪽에 제목을 적고, 시간/제목/둘 다 중 무엇을 보여줄지 정합니다.

밴드 두께 · 글자 크기

슬라이더로 일정 밴드의 두께와 할 일 글자, 제목, 시간 글자 크기를 조절합니다.

5. 알람 & 브라우저 알림

  • 일정을 추가할 때 알람 설정을 켜면, 시작 또는 종료 시각에 알려줍니다.
  • 처음 열면 상단에 "알람을 받으려면 알림을 허용해주세요" 배너가 뜹니다. 허용하기를 누르면 브라우저 알림으로, 허용하지 않으면 화면 안 토스트 메시지와 소리로 알려줍니다.

6. 데이터는 어디에 저장되나요?

💾 모든 일정과 설정은 이 브라우저에만 저장됩니다(서버 전송 없음). 날짜가 바뀌면 그날의 일정은 자동으로 새로 시작되고, 다른 기기나 다른 브라우저와는 동기화되지 않습니다.
